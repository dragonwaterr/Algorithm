# 설계
  1. 두 개의 선거구로 나누기
    -> "1번이 포함된 선거구" vs "그렇지 않은 선거구" 
    -> 조합 활용

  2. 두 개의 선거구에 각각 DFS 로 총 인구수 구하기
    -> 모든 도시를 돌지 못한 경우 (불가능한 방법) 을 체크하기 위해 cnt 체크
  
  3. 두 선거구의 인구차이를 answer 와 비교 후 answer 갱신

  
# 풀고나서 아쉬운 부분

  1. 조합 로직 구현에서 실수
    -> for(int i = idx ~~  로 해놓고서, 이번에 뽑을 인덱스 변수로 i 를 써야하는데 idx 를 써서 한참 동안 발견못함
      --> 변수명 명확하게 다른걸로 구분 필요 
    -> "1번이 포함된 선거구" 의 크기를 1 ~ n-1 까지 뽑는 부분에 불필요한 반복문 하나 더 사용
      --> 실력 이슈
  
  2. 지저분한 부분 
     -> DFS 호출을 여러군데서 하다 보니까, 처음 호출시에 조건체크를 깔끔하게 못했다 
       --> DFS 내에서 DFS 를 재귀호출 할때는 조건체크 제대로 했는데, 최초 DFS 호출 때 빼먹은 부분이 생김
